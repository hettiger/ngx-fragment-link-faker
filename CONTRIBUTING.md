# Contributing

This project is using Gitflow as the Git branching model.
Following configuration applies:

```text
[gitflow "branch"]
        develop = develop
        master = main
[gitflow "prefix"]
        feature = feature/
        release = release/
        hotfix = hotfix/
        versiontag = v
        bugfix = bugfix/
        support = support/
```

> Please be sure to base your pull requests onto the develop branch.
